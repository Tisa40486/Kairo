# Kairo

Kairo est une application de gestion de tâches (Todo List) développée avec **Blazor + ASP.NET Core + C# + SQL**.

## Objectif

Créer une application simple et évolutive permettant de :
- gérer des tâches quotidiennes
- stocker les données en base de données
- utiliser une architecture propre (front / back / data)

## 🛠️ Technologies
- C# (.NET 9+)
- Blazor
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server / SQLite
